LexBench-Browser is maintained as a reproducible benchmark platform. The full policy lives in
the repository root `GOVERNANCE.md`; this page summarizes the parts most contributors need.

## Licenses

- Code: Apache-2.0.
- Documentation and website content: CC-BY 4.0 unless a file states otherwise.
- Dataset licensing: declared in benchmark metadata; do not assume benchmark data inherits the
  code license.

## What can be contributed

- Agent adapters
- Browser backend adapters
- Benchmark task proposals
- Reproducible result submissions
- Evaluation improvements
- Documentation and examples

## Result acceptance

Closed-source agents can appear on the leaderboard. They must disclose agent version, model,
browser backend, benchmark version, judge model, and reproducibility limitations.

Official leaderboard entries require maintainer review and maintainer rerun.

## Contributor credit

Repository contributors may be listed in `CONTRIBUTORS.md`. Paper or technical-report authorship
is handled separately.

See `GOVERNANCE.md` and `EVALUATION_PROTOCOL.md` for the full policy.
