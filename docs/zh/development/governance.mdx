browseruse-agent-bench 按面向浏览器 Agent 的可复现评测框架维护。LexBench-Browser 是内置公开数据集。
完整规则在仓库根目录 `GOVERNANCE.md`；本页总结贡献者最常用的部分。

- 数据集 license：在 benchmark metadata 中单独声明；不要默认继承代码 license。

## 可以贡献什么

- Agent adapter
- Browser backend adapter
- Dataset task proposal
- 可复现 result submission
- Evaluation 改进
- 文档和示例

## 结果接受规则

闭源 Agent 可以上榜，但必须披露 agent version、model、browser backend、benchmark version、
judge model 和 reproducibility limitations。

Official leaderboard entry 需要 maintainer review 和 maintainer rerun。

## 贡献者署名

仓库贡献者可以列入 `CONTRIBUTORS.md`。论文或 technical report authorship 另行决定。

完整规则见 `GOVERNANCE.md` 和 `EVALUATION_PROTOCOL.md`。
